# ScannerDES — Autodiagnóstico de Sostenibilidad (Sello PRO)

Mockup funcional de autodiagnóstico de sostenibilidad basado en los pilares de **Compromiso PRO**, bajo el marco visual del **Sello PRO**.

## Cómo usarlo

Es un **único archivo autocontenido**: abre [`index.html`](index.html) en cualquier navegador (o publícalo con GitHub Pages). No requiere servidor ni instalación.

## Cómo funciona

1. La persona elige **tipo de empresa** (Constructora e Inmobiliaria, Constructora, Inmobiliaria, Especialidades, Industrial, Proveedor) y **tamaño** (Grande / Pequeña-Mediana).
2. El cuestionario muestra **solo los criterios que aplican a ese perfil** (los criterios marcados como "no aplica" en la matriz se excluyen).
3. Cada criterio se evalúa eligiendo el nivel que mejor describe a la empresa, en una escala de **1 a 5 estrellas**.
4. El resultado es el **promedio de los criterios respondidos**, mostrado de forma global y por pilar. Sin recomendaciones (se discuten en taller).

## Notas

- Todo el cálculo ocurre en el navegador; no se guardan ni envían datos.
- Es un mockup de demostración; la versión final/programa se construirá aparte.
